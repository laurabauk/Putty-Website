Izlaz naredbe ls sortira po abecedi i upisuje u datoteku lsout.txt, zajedno sa eventualnim porukama o grešci.

7. Ulazne datoteke za gcc mogu biti:

*.c, *.cc, *.o, *.a

C datoteke izvornog koda *.c, C++ datoteke izvornog koda *.cc, objektne datoteke *.o, arhive objektnog koda (libovi) *.a.

8. U Makefile datoteci... pozivanjem pravila clean:

rm -f *.o *~ a.out $(TARGETS)

brišu se slijedeće datoteke: sve objektne datoteke (*.o), sve privremene datoteke editora (*~) i datoteka a.out.

9. Ukoliko se kod izvršenja naredbe:

13. Program:

#define BUFFSIZE 256
char buf[BUFFSIZE];
int main() {
    int n, fdin, fdout;
    fdout = open("dat.txt", O_WRONLY | O_CREAT, 0644);
    if (fdout == -1) {
        perror("open");
        exit(1);
    }
    fdin = STDIN_FILENO;
    while ((n = read(fdin, buf, BUFFSIZE)) > 0) {
        if (write(fdout, buf, n) != n)
            perror("write");
    }
    if (n < 0)
        perror("read");
    exit(0);
}

Program ima funkcionalnost da sve što korisnik utipka putem tipkovnice upisuje u datoteku dat.txt.
